
# Real Time Image processor 

## -: setup

### Backend

#### Run these command to start the server

- cd into backend
```bash
cd backend 
```
- install dependencies
```bash
npm install 
```
#### to start server in dev  
```bash 
npm run dev
```
#### to start server in prod  
- build the server
```bash 
npm run build 
```
- start the server 
```bash
 npm run start 
```
